Description
---
A classic Tic-Tac-Toe web application built using HTML, CSS, and JavaScript. This interactive browser-based game allows two players to compete against each other (X vs O) on the same device. The game features a clean, colorful interface with a responsive grid layout and automatically tracks player turns until there is a winner or a draw.

Features
---
Two-player mode (X vs O)

Interactive 3×3 game grid

Automatically switches turns between players

Highlights the winning combination

Detects draw situations when the grid is full

Reset button to start a new game instantly

Stylish and responsive design with modern colors

Technologies Used
---
HTML | CSS | JavaScript

Usage

Game Setup:
---
Open the game in a browser. A 3×3 grid will be displayed, ready for two players.
Playing the Game:
Click on empty cells to place your symbol. The game automatically alternates turns and checks for a winning pattern after every move.
Winning & Draw:
If a player aligns three symbols horizontally, vertically, or diagonally, the winning combination is highlighted. If all cells are filled without a winner, the game declares a draw.
New Game:
Press the Reset button to clear the board and begin a fresh match.
